Linux常用命令

发表于 2019-10-31 分类于 操作系统

常用的命令，比如 cat 文件内容查看、find 搜索文件，以及 cut、sort 等管线命令。了解 grep 和 awk 的作用。

man:查看帮助文件

ls：显示文件信息命令

ls-l:显示文件或目录下的详细信息

ls-l c*：所有以c开头的文件

pwd:显示用户当前工作目录

cd：切换目录

mkdir:创建目录

rmdir:删除目录

rm:删除工具

rm –f 文件名 :不显示警告信息直接删除

rm –i 文件名 ：显示警告信息是否删除

rm –r 文件名 ：递归删除

mv 原文件或目录 新文件或目录：更改文件名或移动文件

cp: 文件复制

cat : 显示文件内容

grep:在文件里查找指定字符

shutdown：关机

who：当前用户情况

chmod:文件权限修改

chowm:修改文件拥有者

ps:查看正在执行的进程

kill:向正在执行的进程发送指点的信号

find /etc/ -name passwd ##查找/etc/下名称中带有passwd的文件

cut命令用于显示每行从开头算起 num1 到 num2 的文字。
