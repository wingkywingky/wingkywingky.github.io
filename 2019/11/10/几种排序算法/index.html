几种排序算法

发表于 2019-11-10 分类于 手撕代码

插入排序

  1. 时间复杂度O(n^2)
  2. 稳定排序
  3. 使用于链式结构，也使用于数组
  4. 适合初始基本有序的情况，当初始无序 N较大是不适用

选择排序

  1. 时间复杂度O(n^2)

  2. 不稳定

  3. 使用于链式结构，也使用于数组

冒泡排序

  1. 时间复杂度O(n^2)
  2. 不稳定
  3. 使用于链式结构，也使用于数组
  4. N较大是不适用

快速排序

  1. 时间复杂度O(nlog2n)
  2. 不稳定
  3. 当初始无序 N较大适用

归并排序

  1. 时间复杂度O(nlog2n)
  2. 稳定
  3. 使用于链式结构，也使用于数组
